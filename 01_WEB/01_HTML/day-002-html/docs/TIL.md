## HTML 구조화

| 블럭                 | 인라인                 | 인라인-블럭                     |
| -------------------- | ---------------------- | ------------------------------- |
| div                  | span                   | img                             |
| p, h1~h6, ul, ol, dl | a, strong, em, b, i    | button, input, select, textarea |
| 한 행을 모두 차지    | 글자만큼만 차지        | 글자만큼만 차지                 |
| 높이, 너비 지정 가능 | 높이, 너비 지정 불가능 | 높이, 너비 지정 가능            |

## 자동화

수업과 별개로, 주어진 과제를 다 마무리 하고 120일간 사용할 기록에 대해 자동화 시스템을 만들었다.

#### 방법

1. CLAUDE와 VSCODE의 Copilot Chat을 사용하여 ps1 파일을 생성
2. 터미널 입력으로 정의된 파일과 폴더 구조를 생성

#### 느낀점

- 검증, 디버깅을 시키지 않아도 해준다..
- 여전히 오류는 있다.
